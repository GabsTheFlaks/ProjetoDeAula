# Jules Tutor - Ensino de Programação em C++

Este repositório contém o material prático e os códigos das aulas descritas no nosso Plano de Ensino. O foco deste material é atender pessoas que **nunca tiveram contato com programação**, utilizando **C++** aplicado em problemas reais (como cálculos do dia a dia, médias e "gráficos" desenhados no próprio terminal).

## Módulos do Curso

As pastas deste repositório acompanham o conteúdo programático e contêm exatamente as 5 aulas descritas no plano:

* **[Módulo 1: Lógica Básica](./modulo1_logica)**: Introdução às variáveis, operações matemáticas e tomada de decisão (If/Else) em C++.
* **[Módulo 2: Dados e Gráficos no Terminal](./modulo2_analise)**: Trabalhando com vetores (Arrays), calculando estatísticas simples (média, soma) e plotando gráficos visuais usando caracteres ASCII no terminal.

## Como Executar as Aulas

Como a ideia é focar em alunos do Ensino Médio de Escolas Públicas ou turmas de introdução, recomendamos fortemente o uso do **OnlineGDB** ou **Replit**, pois rodam o C++ direto no navegador e não exigem instalação.

Porém, para testar localmente na sua máquina:
1. Instale um compilador C++ (como o `g++` ou `clang`).
2. Entre na pasta da aula que deseja rodar: `cd modulo1_logica`
3. Compile o arquivo: `g++ aula1_variaveis.cpp -o aula1`
4. Execute: `./aula1`
